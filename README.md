<h1>Currency Exchange Rates Application</h1>

<p>This Java application retrieves currency exchange rates from selected currency exchange offices in Krakow and displays them. It utilizes log4j for logging, JavaFX for the graphical user interface, Scene Builder for designing UI layouts, and web scraping techniques to fetch data from the web.</p>

<h2>Features</h2>

<ul>
    <li>Retrieves currency exchange rates from selected exchange offices in Krakow.</li>
    <li>Displays the retrieved data in a table format within the application.</li>
    <li>Uses log4j for logging messages and errors.</li>
    <li>Utilizes JavaFX for creating a user-friendly graphical interface.</li>
    <li>Designed UI layouts using Scene Builder for better user experience.</li>
</ul>

<h2>Requirements</h2>

<ul>
    <li>Java Development Kit (JDK)</li>
    <li>Apache Maven</li>
    <li>log4j library</li>
    <li>JavaFX library</li>
    <li>Scene Builder</li>
</ul>
